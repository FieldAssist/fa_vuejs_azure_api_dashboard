## ‚ùî Getting Started

### `Azure PAT`

Your Azure Personal Access Token.  
Generate from here: https://dev.azure.com/flick2know/_usersSettings/tokens

Copy it safely to reuse in future.

### `Iteration Path(s)`

The iteration path of backlogs to extract.
